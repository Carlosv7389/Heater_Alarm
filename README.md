# Heater Alarm
A small and easy to mount device that turns on a heater in order to warm a room at a specified time.

## Installation
To clone this repository:

```bash
git clone https://github.com/Carlosv7389/Heater_Alarm
```

Install VScode at [this link](https://code.visualstudio.com/download)
in the "extension" section find platformio 

## Getting Started 
In VScode, open the folder: Plaformio_arduino_wifi

Edit the file include->secrets.h to change the ssid and passwords to your router's ssid and password

Create circuit
